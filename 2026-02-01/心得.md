# 归并算法
```
void merge_sort(int q[],int l,int r){
    if(l>=r)  return;

    int mid=(l+r)/2;
    merge_sort(q,l,mid),merge_sort(q,mid+1,r);
    //先划分

    int k=0;
    int i=l,j=mid+1;
    while(i<=mid&&j<=r){
        if(q[i]<=q[j])  temp[k++]=q[i++];
        else  temp[k++]=q[j++];
    }
    while(i<=mid)  temp[k++]=q[i++];
    while(j<=r)  temp[k++]=q[j++];
    //再合并

    for(i=l,j=0;i<=r;i++,j++){
        q[i]=temp[j];
    }
    //后赋值
}
```
