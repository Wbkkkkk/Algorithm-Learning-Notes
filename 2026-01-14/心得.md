# STL的基本概念
1.STL为何会诞生：为了提高代码的复用性\
2.STL的六大组件：容器，算法，迭代器，仿函数，适配器，空间配置器\
3.迭代器：通过迭代器，访问容器之中的内容   **类似于指针**
# vector的操作
1.`push_back(参数)//尾插一个元素`\
2.`pop_back(无参数)//尾部出去一个元素`\
3.`v.begin()//指向第一个元素`\
4.`v.end()//指向最后一个元素的下一个位置`\
5.`v.rbegin()//指向最后一个元素`\
6.`v.insert()//插入一个元素`\
7.`v.size()//获得长度`\
8.如何声明一个迭代器
```
vector<int>::iterator vBegin=v.begin()
```
# vector的遍历操作
1.通过迭代器遍历
```
for(vector<int>::iterator vBegin=v.begin();vBegin<v.end();vBegin++){
    cout<<*vBegin<<endl;
}
```
2.通过调用容器容量
```
for(int i=0;i<v.size();i++){
    cout<<v[i]<<endl;
}
```
3.高级方法
```
for(auto i:v){
    cout<<i<<endl;
}//不改变v的值
for(auto &i:v){
    i=i*2;
    cout<<i<<endl;
}//会改变v的值
```
