# 二分法
单调序列一定可以二分，但可以二分的不必须是单调序列。\
只要该序列的前段和后端可以由一个性质进行分割，就可以通过二分法找到这个分界点。
## 二分法模板1
```
while(l<r){
    int mid=l+r+1>>1;
    if(check())  l=mid;
    else  r=mid-1;
}
```
因为c++语法中，除以2是下取整，所以l=mid中的mid可能会等于l，导致l=l，进入死循环。
## 二分法模板2
```
while(l<r){
    int mid=l+r>>1;
    if(check())  r=mid;
    else  l=mid+1;
}
```
需要注意，mid的值是需要在循环中更新的，不然也是死循环。
