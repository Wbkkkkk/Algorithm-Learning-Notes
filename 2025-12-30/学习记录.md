# 数组的基础知识
1.数组的内容是连续存放的\
2.数组删除和增加元素大概率要移动其他元素位置**尤其是删除元素，后面的所有元素要向前移动一位**\
3.c++中如何检查一个数组的元素是否连续存放？
```#include <iostream>
using namespace std;

void testarr()
{
    int array[5]={1,2,3,4,5};
    cout<<&array[0]<<" "<<&array[1]<<" "<<&array[2]<<" "<<&array[3]<<" "<<&array[4]<<endl;
    cout<<"地址连续";
}

int main()
{
    testarr();
    return 0;
}
```
